# 🔥 FastOil - บริการมิติใหม่ของการส่งน้ำมันแบบ On-Demand ⚡

**FastOil** คือแอปพลิเคชันที่เปลี่ยนวิธีการเติมน้ำมันของคุณ โดยการนำบริการปั๊มน้ำมันมาส่งใหุ้ถึงที่ ไม่ว่าคุณจะอยู่ที่ไหน ทุกที่ ทุกเวลา! 🚛💨⛽

## 🚀 ฟีเจอร์ที่โดดเด่น
- **Dashboard สุดพรีเมียม**: ติดตามยอดขายและสถิติด้วยกราฟ Glass UI ที่สวยงามและลื่นไหล
- **ระบบติดตามคำสั่งซื้อ (Real-time Tracking)**: ลูกค้าสามารถติดตามสถานะออเดอร์ได้ทันทีผ่าน Widget ที่หน้าจอ
- **ระบบความปลอดภัยเข้มงวด (Security Armor)**: การยืนยันตัวตนด้วย JWT, Role-Based Access และการตรวจสอบข้อมูลเพื่อความปลอดภัย 🛡️
- **Responsive Design**: ใช้งานได้สมบูรณ์แบบบน PC, Mac, iPad และมือถือทุกรุ่น
- **Admin Panel Suite**: ตั้งราคา กรองข้อมูลลูกค้า จัดการสมาชิก และตรวจสอบกิจกรรมย้อนหลังได้ 100% 🔧📊

## 🛠️ รายละเอียดทางเทคนิค (Tech Stack)
- **Frontend**: React (Vite) + Framer Motion (Animations) + Tailwind CSS (Styling) 🎨
- **Backend**: Node.js + Express (API) ⚙️
- **Database**: LibSQL (Turso) ฐานข้อมูลยุคใหม่ที่รวดเร็วและยืดหยุ่น 📉

## ☁️ การติดตั้งและรันระบบ (Deploying to Render)
1. **GitHub**: พุชโค้ดทั้งหมดขึ้นที่ Repo ของคุณ
2. **Render Backend**: เชื่อมต่อ URL ฐานข้อมูล Turso ผ่าน Environment Variables
3. **Render Frontend**: ตั้งค่า Build command `npm run build` และถังเก็บไฟล์ `dist`
